경력 5년 Full Stack E-commerce Domain Scraping / API / Refactoring

자기소개

저는 5년간 다양한 환경에서 웹 및 데스크톱 애플리케이션을 개발해 온 개발자입니다. C#과 JavaScript를 중심으로 .NET Framework, ASP.NET Core, Vue.js를 활용한 서비스 개발 경험을 보유하고 있으며, WinForms와 WPF를 이용한 업무용 프로그램 개발부터 웹 서비스 구축까지 폭넓은 프로젝트를 수행해 왔습니다.

개발 업무를 수행하면서 가장 중요하게 생각하는 것은 안정적인 서비스 제공과 문제 해결 능력입니다. 새로운 기능을 구현하는 것뿐만 아니라 기존 시스템을 분석하고 개선하여 사용자에게 더 나은 경험을 제공하는 데 집중해 왔습니다. 특히 요구사항을 정확하게 이해하고 이를 실제 서비스에 반영하는 과정에서 개발자와 사용자 사이의 간극을 줄이는 것을 중요하게 생각합니다.

백엔드 개발에서는 C# 기반의 .NET Framework 및 ASP.NET Core 환경에서 API와 비즈니스 로직을 구현하였으며, MySQL, MongoDB, Redis를 활용하여 데이터 저장 및 캐시 구조를 설계하고 운영한 경험이 있습니다. 프론트엔드에서는 Vue.js와 JavaScript를 활용하여 사용자 인터페이스를 개발하고, 백엔드와의 연동을 통해 완성도 높은 서비스를 구축하였습니다. 또한 WinForms와 WPF를 이용한 데스크톱 애플리케이션 개발 경험을 통해 다양한 환경에서 요구되는 기술적 문제를 해결해 왔습니다.

저는 기술 변화에 대한 적응력이 높고 새로운 분야를 학습하는 것을 즐기는 개발자입니다. 최근에는 AI 기술이 소프트웨어 개발 환경에 빠르게 적용되고 있는 만큼, 개발 생산성을 높이고 서비스에 새로운 가치를 제공할 수 있는 AI 활용 방안에 관심을 가지고 학습하고 있습니다. 단순히 현재의 기술에 머무르지 않고 지속적으로 성장하며 더 나은 서비스를 만드는 개발자가 되고자 노력하고 있습니다.

앞으로도 다양한 기술 경험과 문제 해결 능력을 바탕으로 조직의 목표 달성에 기여하고, 사용자에게 실질적인 가치를 제공할 수 있는 개발자로 성장해 나가겠습니다.

  • 사이트 RestAPI 통신 패킷 및 취약점 분석을 통해 대량 상품 데이터 수집 기능 고도화
  • 국내 오픈마켓 API 연동 기반의 대량 상품 관리 기능 구현
  • 텍스트 및 이미지 번역 등 데이터 가공 업무 경험 보유

Vue JS, ASP.NET Core를 활용한 이커머스 솔루션 풀스택 개발

  • 외주 개발 레거시 프로젝트의 시스템 분석 및 코드 리펙토링 주도
  • 비효율적이고 재사용이 어려운 하드코딩 시스템을 개선해 생산성과 유지보수성 향상

리펙토링 리스트

  3. 불안정하게 운영되던 Redis session 시스템을 전면 개선해 서비스 안정성 강화
  4. 국내외 쇼핑몰 대상 데이터 스크래핑 작업 다수 수행

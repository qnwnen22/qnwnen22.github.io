  • 수집 금지,제외 단어 추가 기능 개발
  • 해외 상품 자동 번역 기능 구축
  • 환율 자동 계산기 개발

4. 소개
  • 오픈마켓, 소셜커머스에서 상품을 등록/판매하는 셀러들은 상품 등록 시간에 한계가 존재합니다. 상품 수기 등록시 5분 기준 1~2개의 상품밖에 등록 되지 않습니다.
  • CashData는 이러한 한계를 극복 할 수 있는 솔루션을 제공합니다.
  • 해외 오픈마켓에 등록 되어있는 원하는 상품, 혹은 카테고리를 대량으로 수집하여 국내 오픈 마켓에 자동으로 대량 등록 할 수 있습니다.
  • CashData를 사용시 5분 기준 약 3,600개의 상품 등록이 가능해 집니다.
  • 프로그램의 목적상 소수의 회원만 유지중에 있으며 약 100~200명의 회원이 이용중입니다.
  • https://cashdata.modoo.at/?pc=1
